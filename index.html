ENM cmedit commands
  • network_check Check statuses on a node

    cmedit get RBS01 EUtranCellFDD.(operationalState,administrativeState,availabilityStatus,cellBarred) –t

    cmedit get RBS01 ENodeBFunction.* -t

    cmedit get RBS01 ENodeBFunction.(EUtranCellFDDId==RBS01_1,operationalState,administrativeState,availabilityStatus,cellBarred) -t

    cmedit get RBS01 TermPointToENB.(operationalState,administrativeState,availabilityStatus) -t

    cmedit get RBS01 AuxPlugInUnit.(unitType,operationalState,administrativeState,availabilityStatus) -t

  • network_locked Lock/unlock and restart a single node

    cmedit set RBS01 EUtranCellFDD administrativeState=LOCKED

    cmedit set RBS01 EUtranCellFDD administrativeState=LOCKED --force

    cmedit set RBS01 EUtranCellFDD administrativeState=UNLOCKED --force

    cmedit set RBS01 AuxPlugInUnit.(AuxPlugInUnitId==RUS) --force

    cmedit action RBS01 ManagedElement manualRestart.(restartRank=RESTART_WARM;restartReason=PLANNED_RECONFIGURATION;restartInfo=someInfo)

  • find_in_pageFind all nodes

    cmedit get RBS01 EUtranCellFDD.(operationalState==DISABLED,administrativeState==UNLOCKED) –t

  • find_in_pageExport/Import nodes

    cmedit export --ne RBS --filetype 3GPP --filecompression none --jobname myExportJob

    cmedit export --ne SubNetwork=ONRM_ROOT_MO,SubNetwork=RBS --filetype 3GPP --filecompression none --jobname myExportJob

    cmedit export --status --jobname myExportJob

    cmedit export --download --jobname myExportJob
